How to Configure SAML 2.0 for Base CRM

This setup might fail without parameter values that are customized for your organization. Please use the Okta Administrator Dashboard to add an application and view the values that are specific for your organization.

  1. Login to Base CRM as an administrator.

  2. Navigate to Account > Settings > Single Sign On.

  3. Click Configure.

  4. Do the following (see screen shot at end of step for reference):

    • Make a copy of the UUID value.

    • Select the Automatic Setup checkbox.

    • Copy and paste the following Metadata URL:

      Sign into the Okta Admin dashboard to generate this value.

    • Click Save.

  5. Click the Enable button:

  6. In Okta, select the General tab for the Base CRM app, then click Edit.

    • Enter the UUID value that you copied earlier into the corresponding field.

    • Click Save.

  7. Done!


Notes:

SP-initiated flows and IDP-initiated flows are supported.

Just In Time (JIT) provisioning is not supported.

For SP-inititated Flows:

  1. Open the login URL: https://core.futuresimple.com/users/login.

  2. Enter your Email.

  3. Click Log in.
